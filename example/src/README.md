# DEGREE NFT

Backend project to connect a blockchain with an API system and create NFTs.

---

## Requirements

Wallet addresses must be configured in the .ENV

### Scalfold

clean architecture

```
-> contract
  - ABI
-> routes
  - index
  - mint
  - get
-> services
  - mint
  - get
```

---

## Setup

- Check env file credentials `.env`.
- Copy the ABI to the contract folder `./contract`.

## Running the project

    $ npm run start
